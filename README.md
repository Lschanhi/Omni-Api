## 📄 Descrição do Projeto

O **OmniMarket** é uma plataforma de e-commerce do tipo *peer-to-peer* (P2P), desenvolvida com o objetivo de democratizar o comércio online, conectando diretamente vendedores e compradores em um ambiente seguro, intuitivo e escalável.

Diferente de marketplaces tradicionais centralizados, nossa proposta foca em criar um ecossistema onde qualquer usuário pode criar sua própria vitrine virtual e gerenciar seus produtos com facilidade, reduzindo barreiras de entrada para pequenos empreendedores.

### 🎯 Objetivos do TCC

Este projeto tem como finalidade acadêmica demonstrar a aplicação prática dos conhecimentos adquiridos durante o curso técnico, com ênfase em:

*   **Desenvolvimento Full-Stack:** Integração completa entre interfaces modernas (Frontend) e lógica de negócios robusta (Backend).
*   **Arquitetura em Nuvem:** Implementação e hospedagem de serviços utilizando Microsoft Azure.
*   **Integração de Sistemas:** Criação de APIs RESTful para comunicação eficiente entre aplicações web e mobile.
*   **Gestão de Dados:** Modelagem e manipulação segura de dados relacionais.

### 🛠️ Solução Técnica

A plataforma foi construída sobre o ecossistema .NET, garantindo alta performance e segurança. A arquitetura da solução está dividida em:

*   **Backend:** API robusta construída em **C#** e **.NET 9**, responsável por todas as regras de negócio, autenticação e processamento de transações. Utiliza **Entity Framework Core** para uma camada de acesso a dados otimizada.
*   **Frontend Web:** Interface responsiva desenvolvida com **HTML5**, **CSS3** e **JavaScript**, focada na experiência do usuário (UX) e acessibilidade.
*   **Mobile:** Aplicação nativa multiplataforma desenvolvida em **.NET MAUI**, estendendo o alcance do marketplace para dispositivos móveis.
*   **Banco de Dados:** **SQL Server** hospedado no Azure, garantindo integridade e disponibilidade das informações.

### 🚀 Funcionalidades Principais

*   **Gestão de Usuários:** Cadastro seguro, login e perfis personalizados para compradores e vendedores.
*   **Catálogo Dinâmico:** Listagem de produtos com busca avançada e filtros por categoria.
*   **Carrinho de Compras:** Sistema de gestão de pedidos e checkout simplificado.
*   **Painel do Vendedor:** Dashboard para cadastro de produtos, controle de estoque e visualização de vendas.
*   **Comprovante de Venda em PDF:** O OmniMarket gera um comprovante de venda em PDF para auxiliar compradores e vendedores no acompanhamento das transações. O documento possui caráter informativo e não substitui nota fiscal.
*   **Recibos Sob Demanda com Snapshot:** Os recibos são gerados sob demanda a partir dos dados congelados no pedido. Isso evita armazenamento desnecessário de arquivos e garante que o comprovante mantenha as informações da venda no momento da compra.
